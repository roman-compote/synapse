# Open-Source Models

Open-source LLMs offer transparency, customization, and local deployment capabilities. Key advantages include data privacy, cost control, and the ability to fine-tune for specific use cases.

## Popular Open-Source Models
- [[Llama 3]] - Meta's flagship model with excellent reasoning and coding capabilities
- [[Mistral 7B]] - Efficient 7B parameter model with strong performance-to-size ratio
- [[Code Llama]] - Specialized version of Llama optimized for code generation
- [[Falcon]] - UAE's breakthrough model with strong multilingual capabilities  
- [[Vicuna]] - Fine-tuned LLaMA model trained on user conversations

### Local Deployment Tools
- [[Ollama]] - Simplified local model management and inference
- [[LM Studio]] - User-friendly GUI for running local LLMs with model management
- [[Docker LLM]] - Containerized deployment with Docker and Docker Compose stacks
- [[Docker Model Runner]] - Docker Desktop's native model execution (macOS Apple Silicon)
- [[Open WebUI]] - Feature-rich web interface supporting multiple LLM backends
- [[Text-generation-webui]] - Comprehensive web UI with extensive customization options
- [[Jan AI]] - Polished cross-platform desktop app for local AI
- [[Llamafile]] - Single executable file deployment for maximum portability
- [[vLLM]] - High-performance inference engine optimized for production
- [[LocalAI]] - OpenAI-compatible API server for local models

# Reasoning Models (2025)

Specialized models designed for complex problem-solving, mathematical reasoning, and step-by-step analysis with visible thinking processes.

## Latest Reasoning Models
- [[GPT-o3]] - OpenAI's most advanced reasoning with visual capabilities (Apr 2025)
- [[DeepSeek-R1]] - Open-source breakthrough reasoning model via pure RL (~$300k cost)
- [[Qwen3]] - Alibaba's hybrid reasoning model with 119-language support
- [[GPT-o1]] - OpenAI's original reasoning-focused model for complex problems

## Reasoning Model Features
- **Step-by-Step Thinking**: Visible chain of thought processes
- **Mathematical Excellence**: PhD-level performance on complex problems
- **Code Reasoning**: Advanced programming problem solving
- **Multimodal Reasoning**: Text + visual analysis capabilities
- **Extended Processing**: Can work on problems for hours

# Model Selection Criteria

## By Use Case
- **General Chat**: GPT-5, Claude 4 Sonnet, Gemini Pro
- **Coding**: Claude 4 Opus, Code Llama, GPT-5
- **Reasoning**: GPT-o3, DeepSeek-R1, Qwen3, GPT-o1
- **Multilingual**: Qwen3, Falcon, PaLM 2
- **Local/Private**: Llama 3, Mistral 7B, DeepSeek-R1

## By Performance Requirements
- **Speed**: GPT-5 (non-thinking), Claude 4 Sonnet (instant mode)
- **Accuracy**: GPT-o3, Claude 4 Opus, DeepSeek-R1 (reasoning tasks)
- **Cost**: Mistral 7B, Vicuna, open-source variants
- **Context**: Claude 4 Sonnet (1M tokens), Llama 3 (128k)

## By Deployment
- **Cloud API**: GPT-5, Claude 4, GPT-o3
- **Local**: Ollama with Llama 3, Mistral 7B, DeepSeek-R1 distilled
- **Enterprise**: Claude 4 Opus, GPT-5 Pro, custom deployments
- **Mobile**: Qwen3-0.6B, specialized compact models

## By Budget
- **Free Tier**: GPT-5 (limited), Claude 4 Sonnet (limited)
- **Cost Effective**: GPT-5 API, open-source models
- **Premium**: Claude 4 Opus, GPT-o3-pro, extended reasoning
- **Enterprise**: Custom pricing, volume discounts
