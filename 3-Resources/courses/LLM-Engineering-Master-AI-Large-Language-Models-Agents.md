## Course Overview

Comprehensive 8-week program to become an LLM Engineer through building and deploying 8 real-world LLM applications. Master Generative AI, RAG (Retrieval-Augmented Generation), QLoRA fine-tuning, and AI Agents.

**Course Description:**
"Become an LLM Engineer in 8 weeks: Build and deploy 8 LLM apps, mastering Generative AI, RAG, LoRA and AI Agents."

**Learning Philosophy:**
Hands-on, project-based learning with 8 substantial projects. Experiment with over 20 groundbreaking models and master state-of-the-art techniques. No advanced math required - no calculus or linear algebra needed.

## Course Structure

### 8-Week Program with 8 Projects

#### **Week 1: Web Scraping & Brochure Generation**
**Project 1:** AI-Powered Brochure Generator
- Intelligent website scraping and navigation
- Automated content extraction
- Professional document generation
- Multi-page web scraping capabilities

**Learning Focus:** First business Gen AI product, web automation, structured output

---

#### **Week 2: Frontier APIs & Customer Service**
**Project 2:** Multi-Modal Customer Support Agent
- Airline customer service chatbot
- Multi-modal capabilities (text, images, audio)
- Function calling / tool use
- Professional UI development

**Learning Focus:** Frontier APIs (OpenAI, Claude, Gemini), multi-modal interaction, agent architecture

---

#### **Week 3: Audio Processing**
**Project 3:** Meeting Minutes Generator
- Audio transcription
- Action item extraction
- Meeting summarization
- Open-source and closed-source models

**Learning Focus:** Audio AI, model comparison, practical business automation

---

#### **Week 4: Code Optimization**
**Project 4:** Python to C++ Optimizer
- AI-assisted code conversion
- Performance optimization (60,000x speedup potential!)
- Cross-language translation
- Compiler optimization techniques

**Learning Focus:** AI for code generation, performance engineering, practical optimization

---

#### **Week 5: RAG Systems**
**Project 5:** AI Knowledge Worker
- Company knowledge expert using RAG
- Vector database integration
- Document processing and chunking
- Conversational memory

**Learning Focus:** RAG architecture, vector embeddings, production knowledge systems

---

#### **Week 6-8: Capstone - Three-Part Price Prediction System**

**Project 6 (Capstone Part A):** Frontier Model Price Prediction
- Product price prediction from short descriptions
- Frontier model implementation (GPT-4, Claude, etc.)
- Baseline performance establishment

**Project 7 (Capstone Part B):** Fine-Tuned Open-Source Model
- QLoRA fine-tuning
- Open-source model optimization
- Competing with frontier model performance
- Cost-effective alternative

**Project 8 (Capstone Part C):** Autonomous Multi-Agent Deal Finder
- Multi-agent collaboration
- Deal spotting and notification system
- Agent orchestration
- Real-world autonomous application

**Learning Focus:** Complete ML lifecycle - baseline, fine-tuning, deployment, autonomous operation

## Tools & Technologies Covered

### LLM Providers (20+ Models)
- **OpenAI** - GPT-4o, GPT-4, GPT-3.5
- **Anthropic** - Claude 3.5 Sonnet, Claude 3 family
- **Google** - Gemini Pro, Gemini Flash
- **Open-Source** - Llama, Mistral, and others
- **Local** - Ollama for local deployment

### AI/ML Frameworks
- **LangChain** - LLM application framework
- **Vector Databases** - ChromaDB, FAISS, or Pinecone
- **Fine-Tuning** - QLoRA, LoRA techniques
- **Audio Processing** - Whisper and alternatives

### Development Tools
- **Python** - Primary language
- **Jupyter Notebooks** - Interactive development
- **Web Scraping** - BeautifulSoup, requests
- **UI Frameworks** - Gradio or Streamlit
- **APIs** - RESTful integration patterns

- **Cloud Platforms** - Deployment patterns
- **Production Patterns** - Scalability and monitoring
